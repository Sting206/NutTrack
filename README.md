# Дневник питания

Приложение для отслеживания питания, водного баланса и замеров тела.

## Возможности
- 📊 Отслеживание калорий, белков, жиров, углеводов
- 💧 Контроль водного баланса
- 📏 Замеры тела и веса
- 📅 История питания
- 🔄 Синхронизация с Google Sheets

## Технологии
- HTML5
- CSS3 (градиенты, анимации)
- JavaScript
- Chart.js
- Google Sheets API
